# Tea-Time-Watch
We can say this is a web app like Youtube.

# Initialization of git and npm
```
git init
npm init
```

# Directory setup 
```
mkdir public
mkdir temp
type nul > .gitkeep
```
```
type nul > .env
type nul > .env.sample

mkdir src
```

# Started Package Installation

```
npm i -D nodemon
npm i nodemon
```
```
type nul > app.js
type nul > index.js
type nul > constants.js
```
